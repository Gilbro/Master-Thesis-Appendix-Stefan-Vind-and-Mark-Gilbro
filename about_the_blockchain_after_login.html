The blockchain or timestamp server as it is called by Satoshi Nakomoto, is a bi-product of the bitcoin, while it was not the first thing for that was the focus when the whitepaper was release it is now known as one of the key innovations of the bitcoin.


The blockchain is a public ledger of all transactions ever made on the network. It is constantly growing as completed blocks are added, the blocks in a blockchain are added chronological so each node in the network will get a copy of the blockchain, and will because a blockchain is a series of blocks ‘chained’ together, it will be possible to track it back to the first block.
